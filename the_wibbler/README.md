# the-wibbler
Create a random wibbly wobbly mess in .gif form

## Getting Started

### Prerequisites
Python modules required:
```
PIL
glob
matplotlib
random
scipy
math
```

### Installing
Download and run the scripts in this order:
1) the_wibbler.py
2) gif_make.py

### Examples
Check the example folder in root wibbler directory

## Built with
Python 3.8 (on Anaconda/Spyder)
